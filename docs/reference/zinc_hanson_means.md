# Example founder means from ZINC Hanson experiment

This dataset contains founder mean frequencies for the ZINC Hanson
experiment.

## Usage

``` r
zinc_hanson_means
```

## Format

A tibble with 3,987,264 rows and 6 variables:

- chr:

  Chromosome

- pos:

  Genomic position (bp)

- TRT:

  Treatment (C or Z)

- REP:

  Replicate number

- founder:

  Founder strain

- freq:

  Allele frequency
